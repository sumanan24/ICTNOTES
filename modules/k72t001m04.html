Digital Electronics and Computer Architecture

Module Code: K72T001M04 / Level 05

Compulsory Relevant Unit: All Units Pre-Requisites: K72T001M03 75 Notional Hours 42 Theory 18 Practical 15 Self-study

Learning Outcomes

General Objective

At the end of this module, students will understand logic gates, describe basic computer components and functional organization, describe processor organization, and know software systems and IoT integration.

Specific Objectives

At the end of this module the student shall be able to:

  1. Demonstrate an understanding of logic gates
  2. Design a logic circuit for a given scenario

Learning Content / Topics

1. Logic Gates (12 hrs.)
2. Computer Organization and Architecture (21 hrs.)
3. Von-Neumann architecture (3 hrs.)
4. Beyond Von-Neumann architecture (6 hrs.)
