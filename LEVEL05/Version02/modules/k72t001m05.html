Object Oriented Programming Concepts

Module Code: K72T001M05 / Level 05

Compulsory Relevant Unit: All Units Pre-Requisites: None 100 Notional Hours 54 Theory 36 Practical 10 Self-study

Learning Outcomes

General Objective

At the end of this module, students will understand the basic program structures (Sequence, Selection, Iteration) and understand the syntax of selected programming language.

Specific Objectives

At the end of this module the student shall be able to:

  1. Draw flowcharts and write pseudocodes
  2. Write relevant program codes
  3. Test the developed program/s
  4. Apply IDEs for program development
Module Aim

To enable students to:

  • Understand the basic program structures (Sequence, Selection, Iteration)
  • Understand the syntax of selected programming language

Learning Content / Topics

1. Basics of computer programming (18 hrs.)
2. Introduction to Object-Oriented Programming (6 hrs.)
3. OOP concepts (24 hrs.)
4. Association, aggregation and composition (6 hrs.)
