Mathematics for Computing

Module Code: K72T001M03 / Level 05

Compulsory Relevant Unit: All Units 25 Notional Hours 20 Theory 0 Practical 5 Self-study

Learning Outcomes

General Objective

At the end of this module, students will understand different types of number systems, know conversion between different bases, acquire skills in calculations using different number bases, and understand basic statistics concepts and computer coding systems.

Specific Objectives

At the end of this module the student shall be able to:

  1. Number systems classification and conversion techniques
  2. Arithmetic operations in different number bases
  3. Fractional number representation methods
  4. Set theory concepts and applications
  5. Statistical measures calculation
  6. Computer coding systems identification and usage

Learning Content / Topics

1. Types of Number Systems (01 hrs.)
2. Conversion between different bases (06 hrs.)
3. Calculation using different number bases (04 hrs.)
4. Fractional number representation (02 hrs.)
5. Sets theory (02 hrs.)
6. Statistics fundamentals (03 hrs.)
7. Computer Codes (02 hrs.)
